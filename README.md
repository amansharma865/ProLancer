# ProLancer

**ProLancer** is a modern freelancing marketplace web application that connects clients and freelancers. It allows freelancers to showcase their skills through gigs and enables clients to browse, sort, and hire professionals seamlessly. Designed with an intuitive interface, real-time chat, and review system, ProLancer simplifies freelance hiring through a smooth and engaging user experience.

## Features
- **Dual Roles:** Separate user experiences for freelancers and clients.  
- **Gig Creation:** Freelancers can create detailed gigs with descriptions, pricing, and categories.  
- **Gig Browsing:** Clients can explore and sort gigs based on categories, skills, or ratings.  
- **Real-Time Chat:** Instant messaging between freelancers and clients for smooth communication.  
- **Review System:** Clients can review freelancers after completing a project.  
- **User Authentication:** Secure signup and login functionality for both roles.  
- **Responsive UI:** Fully optimized for desktop, tablet, and mobile devices.  

---

## How It Works

1. **Freelancer Registration:**  
   - Freelancers sign up and create gigs showcasing their skills and experience.  
2. **Client Browsing:**  
   - Clients explore gigs using filters like category, skillset, or price range.  
3. **Hiring Process:**  
   - Clients hire freelancers based on their gig details and reviews.  
4. **Real-Time Communication:**  
   - Built-in chat enables direct interaction for project discussion.  
5. **Feedback and Reviews:**  
   - After completion, clients can rate and review freelancers.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Token)  
- **Real-Time Chat:** Socket.io  
- **Deployment:** Vercel
